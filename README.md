# Roll and Ball

¡Bienvenido a **Roll and Ball**! Un emocionante juego en 3D donde controlarás una bola y deberás superar desafiantes niveles llenos de obstáculos.

## Descripción

**Roll and Ball** es un juego de plataformas donde tu objetivo es guiar una bola a través de una serie de niveles, recogiendo bloques y evitando enemigos. 
Con cada nivel, la dificultad aumenta, ofreciendo una experiencia divertida y desafiante.

## Características

- **Gráficos atractivos**: Disfruta de un entorno 3D colorido y vibrante.
- **Niveles variados**: Supera diferentes niveles con distintos desafíos y mecánicas.
- **Controles intuitivos**: Flechas del teclado, y space para saltar.

## Requisitos

- Sistema operativo: Windows / macOS / Linux
- Memoria RAM: 4 GB (mínimo)
- Tarjeta gráfica: Compatible con DirectX 11
- Espacio en disco: 500 MB

## Instalación

1. Descarga el juego desde la [# Roll and Ball](https://github.com/Danipr2425/Roll-a-Ball)
2. Extrae el archivo ZIP en tu computadora.
3. Abre la carpeta extraída y ejecuta el archivo `RollAndBall.exe` (o el equivalente en tu sistema operativo).

## Cómo jugar

1. Usa las teclas de flecha o WASD para mover la bola.
2. Recoge bloques para aumentar tu puntuación.
3. Evita los obstáculos y completa cada nivel en el menor tiempo posible.

¡Diviértete jugando a **Roll and Ball**!
